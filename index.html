This project is a color guessing game where the player guesses the specified color from a set of six randomly generated colors.

Weather App

This is a weather app created to display realtime weather data for a list of cities specified by the user.

The list of cities is stored and retrieved from the browser's local storage on load and when the user updates the list. It also features the ability to toggle dark mode.

Weather data is found using OpenWeatherMap API.
